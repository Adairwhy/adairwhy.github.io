解决npm WARN config global --global, --local are deprecated. Use --location=global instead.

将npm升级到最新版本即可
升级方法

  1. 在windows中以管理员身份打开cmd，然后执行命令
    npm install -g npm-windows-upgrade
  2. 更改脚本策略
    下载Windows Power Shell
    然后以管理员身份运行，执行命令
    set-ExecutionPolicy RemoteSigned
    输入Y
    成功更改脚本策略
  3. 在Windows Power Shell上运行命令
    npm-windows-upgrade

问题解决！
其实此问题是由版本号导致

解决Node.js项目报错SyntaxError: Unexpected token …

Node.js版本问题

  1. 安装Node.js版本管理工具
    npm install -g n
  2. 安装最新稳定版 node
    npm lts
  3. 使用 n 切换版本
    n
  4. 查看Node.js版本
    如果不成功，可能是Node安装路径问题
  5. 启动Node.js服务
    forever start xxx.js
